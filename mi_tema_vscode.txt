Aquí puedes guardar tu configuración de colores de VSCode para llevarla a otros IDEs.

Simplemente copia las secciones "workbench.colorCustomizations" y "editor.tokenColorCustomizations"
de tu archivo settings.json y pégalas aquí dentro.

Ejemplo de cómo se vería:

{
  "workbench.colorCustomizations": {
    "activityBar.background": "#282c34",
    "titleBar.activeBackground": "#21252b",
    "titleBar.activeForeground": "#c8c8c8",
    "editor.background": "#282c34",
    "editor.foreground": "#abb2bf",
    "editorCursor.foreground": "#528bff",
    "sideBar.background": "#21252b",
    "list.hoverBackground": "#2c313a"
    // ... y muchos más colores
  },
  "editor.tokenColorCustomizations": {
    "textMateRules": [
      {
        "scope": "comment",
        "settings": {
          "foreground": "#5c6370",
          "fontStyle": "italic"
        }
      }
      // ... y muchas más reglas de sintaxis
    ]
  }
}
